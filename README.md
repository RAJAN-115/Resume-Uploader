# Resume Uploader - MERN Stack Application

A full-stack web application built with the MERN (MongoDB, Express.js, React.js, Node.js) stack for managing and uploading resumes.

## 🚀 Features

- Resume upload and management
- User authentication
- Modern Material-UI interface
- RESTful API architecture
- File storage and management
- Responsive design

## 📁 Project Structure

```
resume-uploader/
├── resumeuploaderreactjsui/     # Frontend React application
│   ├── src/                     # Source files
│   ├── public/                  # Static files
│   └── package.json            # Frontend dependencies
│
└── resumeuploaderexpressapi/    # Backend Express application
    ├── controllers/            # Route controllers
    ├── models/                 # Database models
    ├── routes/                 # API routes
    ├── middlewares/           # Custom middlewares
    ├── config/                # Configuration files
    └── package.json           # Backend dependencies
```

## 🛠️ Tech Stack

### Frontend

- React.js
- Material-UI
- Redux Toolkit
- React Router
- Axios

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer (for file uploads)
